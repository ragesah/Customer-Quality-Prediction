There are six folders:
1) dataset - contains training, validation dataset. Any scripts related to data-preprocessing.

2) figures - includes figure generated during EDA

3) model - contains code related to model building

4) util - contains files related to packages and processing

5) result - include prediction result testingCandidate.csv column name 'responded'

6) tool - has master executable files(jupyter notebooks) which are the main file to be executed.

Note:
--------  
execute "customer_quality_prediction.ipynb" for Customer Quality Prediction
execute "time_series_forecast.ipynb" for Predicting number of failures
